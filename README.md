# Responsive Login & Registration Form

Um componente de formulário de autenticação (Login e Cadastro) moderno, limpo e totalmente responsivo desenvolvido com HTML5 e CSS3. O projeto apresenta uma interface intuitiva focada na experiência do usuário, utilizando uma paleta de cores equilibrada e elementos geométricos bem definidos, ideal para sistemas web, dashboards ou telas de acesso iniciais.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do formulário, campos de entrada (`<input type="email">`, `<input type="password">`) e links de navegação.
* CSS3: Estilização completa, uso de bordas suavizadas, gerenciamento de sombras projetadas e definição de estados.
* Flexbox: Centralização absoluta do card na tela e organização do fluxo dos elementos internos verticalmente.

## Funcionalidades e Técnicas Aplicadas

* Campos de Entrada Estilizados: Caixas de texto com cantos arredondados, bordas sutis e placeholders claros ("Enter your email" e "Enter your password") que garantem excelente legibilidade.
* Hierarquia Visual Clara: Título de destaque centralizado acompanhado de links utilitários estrategicamente posicionados, como o redirecionamento de "Forgot password?".
* Botão de Ação de Alto Impacto: Botão "Login" personalizado com preenchimento em verde esmeralda sólido de alto contraste, alinhando-se perfeitamente com a identidade visual proposta.
* Link de Alternância de Fluxo: Rodapé do card contendo uma chamada para usuários não registrados ("Don't have an account? Signup"), permitindo a transição intuitiva para a tela de cadastro.
* Efeito de Profundidade: Aplicação de cantos suavizados com `border-radius` e uma sombra (`box-shadow`) discreta sob o card branco, destacando-o sobre o plano de fundo verde texturizado.
